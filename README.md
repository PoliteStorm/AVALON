## 🚀 **GETTING STARTED**

### **📋 Prerequisites:**
- Python 3.8+
- NumPy, SciPy, Pandas
- FFmpeg (for MP3 conversion)
- Validated fungal data CSV

### **🔧 Installation:**
```bash
# Clone the repository
git clone https://github.com/PoliteStorm/AVALON.git
cd AVALON

# Install dependencies
pip install numpy scipy pandas matplotlib

# Install FFmpeg (system dependent)
# Ubuntu/Debian: sudo apt install ffmpeg
# macOS: brew install ffmpeg
# Windows: Download from ffmpeg.org
```

### **🎯 Quick Start:**
```bash
# Run basic fungal computing system
python3 enhanced_fungal_computing_system.py

# Run real-time mushroom recorder
python3 real_time_mushroom_recorder_simple.py
```
